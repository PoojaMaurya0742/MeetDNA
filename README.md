MeetDNA is an AI-powered meeting intelligence platform that transforms conversations into structured organizational memory.

Instead of simply recording meetings, MeetDNA automatically transcribes discussions, extracts important decisions, action items, risks, and insights, then stores them using persistent memory for future recall.

Key Features:
✅ Real-time video conferencing
✅ AI speech transcription
✅ Automatic meeting summaries
✅ Decision & action item extraction
✅ Persistent memory with Hindsight
✅ Meeting history and semantic search
✅ AI-generated meeting briefings
✅ PDF summary export
