# 🧬 MeetDNA

### Transforming Meetings into Organizational Memory

**MeetDNA** is an AI-powered meeting intelligence platform that automatically captures conversations, extracts key decisions, identifies action items, generates intelligent summaries, and stores valuable organizational knowledge using persistent memory.

By combining real-time collaboration, AI-driven insight extraction, Retrieval-Augmented Generation (RAG), and Hindsight-powered memory, MeetDNA helps teams reduce information loss, improve meeting productivity, and make better decisions over time.

---

## 🚀 Key Features

### 🎥 Real-Time Video Meetings

* Create and join meetings using secure meeting codes
* High-quality video and audio communication
* Screen sharing support
* Live participant management

### 📝 AI-Powered Live Transcription

* Real-time speech-to-text conversion
* Automatic transcript generation
* Noise and filler-word filtering
* Continuous meeting documentation

### 🧬 DNA Strands Intelligence Engine

MeetDNA automatically extracts and categorizes:

* ✅ Decisions
* ✅ Action Items
* ✅ Risks
* ✅ Insights
* ✅ Context

This transforms unstructured conversations into actionable business intelligence.

### 🧠 Persistent Memory with Hindsight

* Long-term memory storage
* Historical meeting recall
* Context-aware retrieval
* Cross-meeting knowledge retention

### 🔍 RAG-Based Knowledge Search

* Semantic search across previous meetings
* Natural language querying
* AI-generated contextual briefings
* Instant access to historical decisions

### 📄 Automated Meeting Summaries

* Executive summaries
* Decision tracking
* Action item assignment
* Risk identification
* Meeting outcome reports

### 📥 PDF Export

* Download meeting summaries instantly
* Share reports with stakeholders
* Maintain organizational records

## 🔄 Workflow

### 1. Authentication

Users securely log in using JWT-based authentication.

### 2. Meeting Creation

Hosts create meetings and receive a unique meeting code.

### 3. Participant Collaboration

Participants join meetings using the generated code.

### 4. Live Intelligence Capture

Audio is transcribed and processed in real time.

### 5. Insight Extraction

AI identifies decisions, actions, risks, insights, and context.

### 6. Memory Storage

Important knowledge is stored using Hindsight and ChromaDB.

### 7. Summary Generation

Comprehensive meeting reports are generated automatically.

### 8. Intelligent Recall

Future meetings leverage historical context for better decision-making.

---

## 💡 Why MeetDNA?

Traditional meeting platforms record conversations.

MeetDNA remembers them.

Instead of creating isolated transcripts, MeetDNA builds a continuously growing organizational memory system that helps teams retain knowledge, improve collaboration, and make informed decisions using historical context.

---

## 🎯 Problem Statement

Organizations conduct hundreds of meetings every month, yet valuable information is often lost after conversations end.

MeetDNA solves this challenge by transforming meeting discussions into searchable, structured, and persistent knowledge that can be recalled whenever needed.
